# Iterated-prisonners-dilemma
Modular prisoner's dilemma engine in Java

## Running
1) To compile:
```sh
    javac src/Main.java
```
2) To run:
```sh
    java src/Main
```
3) To clean:
```sh
    rm src/*.class
```
